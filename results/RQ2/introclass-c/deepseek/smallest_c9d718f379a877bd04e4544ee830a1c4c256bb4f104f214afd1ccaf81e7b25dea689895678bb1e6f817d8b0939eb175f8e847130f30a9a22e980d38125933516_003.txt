/* In the fixed function, we only print out the smallest number that we find.
We can't print out that a number is the smallest if it's not the smallest one.
So we use a variable smallest to store the smallest number we found.
The if statements are used to check whether the number is the smallest.
If it is, we update the smallest to that number. At the end of the program, we print out the smallest number. */

int main(){
	double num1, num2, num3, num4;
	printf("Please enter 4 numbers separated by spaces > ");
	scanf("%lf%lf%lf%lf",&num1,&num2,&num3,&num4);
	double smallest = num1;
	if (num2<smallest) smallest = num2;
	if (num3<smallest) smallest = num3;
	if (num4<smallest) smallest = num4;
	printf("%.0lf is the smallest\n",smallest);
	return 0; 
}
